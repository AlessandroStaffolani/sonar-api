# Sonar Lib

Sonar lib - a Sonarqube api wrapper

## Install

To install 

### Install - Dev

The dev install is only required if additional development is needed for this library

```bash
pip install -e '.[dev]'
```

### Build the application

Run the command:

```
python -m build
```

### Test the application

Run the test with command:

```
pytest
```
